# 🌿 GreenCred

GreenCred is an AI-powered platform that tracks and rewards eco-friendly actions.

## 🚀 Features
- Google Authentication
- AI Image Verification
- Eco Score System
- Leaderboard Ranking
- Clean Dashboard UI

## 🧠 Tech Stack
- Django
- Tailwind CSS
- AI Image Recognition (HuggingFace / Roboflow)

## 🎯 Goal
To make sustainability measurable, rewarding, and engaging.
